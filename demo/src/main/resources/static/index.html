테스트 소개 페이지

간단하고 깔끔한 HTML/CSS 기반 소개 페이지 예시입니다. 이 파일 하나로 바로 퍼블리싱 가능합니다.

목표

이 테스트 페이지는 기능 소개와 주요 흐름을 빠르게 확인하기 위해 설계되었습니다. 반응형이며 접근성을 고려했습니다.

사용 방법

이 HTML 파일을 웹 서버에 올리거나 로컬에서 열어 확인하세요. 수정은 주석을 참고하여 간단히 가능합니다.

주요 기능

반응형 그리드, CTA(행동유도 버튼), 심플한 스타일 시스템을 포함합니다. 필요에 따라 색상과 내용을 바꿔 사용하세요.

확장하기

폼, 모달, 이미지 갤러리 등을 추가하면 실제 퍼블리싱용 소개 페이지로 확장하기 쉽습니다.
